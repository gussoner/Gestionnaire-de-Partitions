# 🎸 Gestionnaire de Partitions & Setlists

Une extension universelle (Firefox / Chrome) conçue pour les musiciens afin d'organiser leurs partitions PDF et de les afficher en plein écran avec un menu vertical discret à droite.

## ✨ Fonctionnalités
- **Chargement local :** Chargez le dossier de votre groupe en un clic.
- **Organisation de Setlist :** Glissez-déposez pour réordonner vos morceaux, enregistrez et modifiez vos playlists.
- **Visionneuse Live :** Mode plein écran avec index rétractable à gauche et commandes de navigation calées verticalement tout à droite pour ne pas empiéter sur la lecture.
- **Auto-adaptatif :** Fonctionne aussi bien si vos PDF sont rangés dans des sous-dossiers par morceau ou s'ils sont tous en vrac dans un dossier unique.

## 🚀 Installation en local (Mode Développeur)

### Sur Firefox
1. Ouvrez Firefox et tapez `about:debugging` dans la barre d'adresse.
2. Cliquez sur **"Ce Firefox"**.
3. Cliquez sur **"Charger un module temporaire..."**.
4. Sélectionnez le fichier `src/manifest.json` de ce projet.

### Sur Chrome / Edge / Brave
1. Ouvrez l'onglet des extensions (`chrome://extensions/`).
2. Activez le **"Mode développeur"** (en haut à droite).
3. Cliquez sur **"Charger l'extension non empaquetée"**.
4. Sélectionnez le dossier `src/` de ce projet.
